### Prerequisites

- Node 16
- MongoDB 5

### Installation

1. Install NPM packages

   ```sh
   npm install
   ```

1. Rename or copy the `.env.dev` file to `.env`
1. Set your database credentials in your `.env` file

## Running the app

```bash
# development
$ npm run start
```

## Build the app

```bash
# development
$ npm run build
```

## Test

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:cov
```
